# Spotify-Clone
A frontend clone of Spotify focused on UI design

This project is a frontend clone of Spotify, designed to replicate its user interface and overall layout.
The goal of this project is to practice frontend development, improve layout structuring, and build a responsive design.

🚀 Features
🎧 Spotify-inspired user interface
📱 Responsive design
🎨 Clean and modern layout
🧩 Structured components
🛠️ Tech Stack
Frontend development
design principles
